# Instructions

## SGP4 Library

1. Clone the [sgp4 library](https://github.com/dnwrnr/sgp4/) into the `Arduino/libraries` directory
2. Inside the `sgp4` library, rename the directory `libsgp4` -> `src`
3. Copy the `library.properties` file in this directory to the `sgp4` directory

## ArduinoJson Library

1. Install version 7.4.3 of the [ArduinoJson](https://github.com/bblanchon/ArduinoJson) library using the Arduino Library Manager

## AzimuthElevation file

1. Download [`AzimuthElevation.h`](https://raw.githubusercontent.com/thiagohersan/Satellite-Tracking-Station/refs/heads/main/04%20cpp/AzimuthElevation/AzimuthElevation.h) to this directory
